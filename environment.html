環境構築

環境構築には、コンパイラとエディタが必要です。以下の表を参考に、自分のOSに合わせたコンパイラとエディタをインストールしてください。

準備知識

MinGW, Command Line Tools for Xcode とは？

MinGW は、Windows 上で C 言語のコンパイルを行うための無料ツールです。

→ MinGW (Wikipedia)

Xcode は、macOS に標準で用意されている統合開発環境です。

→ Xcode (Wikipedia)
